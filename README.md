## Getting Started

### Local Installation

1. Clone the repo
   ```sh
   git clone https://github.com/FoundationScholarship/social-media-clone.git
   ```
2. Install NPM packages
   ```sh
   npm install i
   ```
3. Starting a Development Server
   ```sh
   npm run dev
   ```
